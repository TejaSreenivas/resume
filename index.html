Projects

Advanced Match scheduler

Designed advanced match scheduler using machine learning to factor in various parameters like the probability of raining for a given day, match turnout, etc. while simultaneously addressing constraints based on the type of schedule requested. Using OR-tool and python.

Skin Cancer detection

Apart from training a deep convolutional neural network for identifying skin cancer from an image of a lesion, I have incorporated the capability to accurately identify the exact lesion region within the image without the need to retrain the model.

Unsupervised Data Quality Assessment

I programmed a Data Quality Assessment tool with the primary focus on using unsupervised learning techniques. Also, the program is designed in a way to incorporate ad-hoc models, making the program scalable for incorporating different models with minimal changes to source code.

Context Extraction using Deep Learning

I have designed a language representation of Word Embedding using convolutional and recurrent neural networks (deep learning architectures), which could extract context. This representation can be used in various Natural Language Processing applications, including identifying categories of domains from a small set of words.

Country Club Website

As part of a team project in my master's, we have designed a country club website with MongoDB as database storage. My contribution includes creating entities, reading and writing data into MongoDB, and designing search and filter with pagination using React-js as frontend. Further, I have converted the project into a docker container suitable for deploying on scalable cloud services such as Azure and AWS.

Airbnb database

Reverse engineered the Airbnb website to design a SQL based database and incorporated best practices and normalization techniques to design an efficient database. Also created different views for the database using SQL queries addressing individual webpage requirements of the Airbnb website.
